Seletores CSS

Bem-vindo!

No CSS nós possuímos os seletores por ID

  • Seletores de Tag/Tipo: seleciona os elementos pelo nome da tag. Exemplo: p, h1, div
  • Seletores de Id: seleciona um elemento específico pelo seu ID. Exemplo: #meuId
  • Seletores de Classes: seleciona elementos que possuem uma classe específica. Exemplo: .minhaClasse
  • Seletores Universais: seleciona todos os elementos. Exemplo: *
  • Seletores de Atributo: seleciona elementos com um atributo específico. Exemplo: [type="text"]
