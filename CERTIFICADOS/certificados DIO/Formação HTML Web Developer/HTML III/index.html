HTML – Linguagem de Marcação

O HTML (HyperText Markup Language) é a linguagem base da web, responsável por estruturar conteúdos como textos, imagens, links e formulários.

O que é HTML?

HTML utiliza tags para representar semanticamente cada parte do conteúdo, permitindo que navegadores, leitores de tela e mecanismos de busca compreendam melhor a informação.

  • Estrutura páginas web
  • Trabalha em conjunto com CSS e JavaScript
  • É mantido pelo W3C

Semântica

A semântica no HTML moderno melhora a organização do código e a acessibilidade. Tags como <header>, <main>, <article> e <footer> dão significado ao conteúdo.

Acessibilidade

Um site acessível permite que pessoas com deficiência naveguem utilizando tecnologias assistivas, como leitores de tela. O uso correto de semântica, atributos aria e contraste adequado são fundamentais.
