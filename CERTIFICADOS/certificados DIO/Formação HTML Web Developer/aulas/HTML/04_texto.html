Exemplo de Texto em HTML

Este é um parágrafo simples em um documento HTML. Ele demonstra como o texto pode ser estruturado e exibido em uma página web.

Subtítulo

Você pode usar diferentes elementos para formatar o texto, como:

  • Negrito para destacar palavras importantes.
  • Itálico para enfatizar certas partes do texto.
  • Sublinhado para indicar links ou termos especiais.

Além disso, você pode criar listas ordenadas:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item
