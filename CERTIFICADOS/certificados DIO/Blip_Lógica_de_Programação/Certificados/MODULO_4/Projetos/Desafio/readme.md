# 2️⃣ Calculadora de Partidas Rankeadas

## Descrição
Neste desafio, você deve criar uma função que recebe como parâmetro a quantidade de vitórias e derrotas de um jogador em partidas rankeadas e calcula o saldo de vitórias (vitórias - derrotas). Com base no número de vitórias, a função deve determinar o **nível do herói** conforme a tabela abaixo:

- Vitórias < 10 → Ferro  
- Vitórias entre 11 e 20 → Bronze  
- Vitórias entre 21 e 50 → Prata  
- Vitórias entre 51 e 80 → Ouro  
- Vitórias entre 81 e 90 → Diamante  
- Vitórias entre 91 e 100 → Lendário  
- Vitórias ≥ 101 → Imortal

## Tecnologias e Conceitos Utilizados
- **Variáveis**: para armazenar vitórias, derrotas, saldo e nível do herói.  
- **Operadores**: para cálculos (`-`) e comparações (`<`, `<=`, `>=`).  
- **Estruturas de decisão**: `if...else if...else` para determinar o nível do herói.  
- **Funções**: `calcularNivel(vitorias, derrotas)` retorna a mensagem final.  
- **Laços de repetição**: permite processar múltiplos jogadores até o usuário decidir parar.
