# Projeto Banco de Dados - Oficina Mecânica

Este projeto tem como objetivo a modelagem e implementação de um banco de dados relacional
para o contexto de uma oficina mecânica.

O projeto contempla:
- Modelagem lógica baseada em modelo ER
- Criação do esquema relacional
- Implementação do banco em PostgreSQL
- Inserção de dados para testes
- Consultas SQL simples e avançadas utilizando:
  - SELECT
  - WHERE
  - JOIN
  - ORDER BY
  - GROUP BY	override
  - HAVING
  - Atributos derivados
